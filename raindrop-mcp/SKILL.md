# Raindrop.io MCP Server

Connect Raindrop.io to your AI assistant with a simple MCP server. Use it to organize, search, and manage bookmarks with natural language.

## Capabilities

- **Bookmark Management**: Create, update, and delete collections and bookmarks.
- **Search & Filtering**: Advanced search by tags, domain, type, date, and more.
- **Tag Management**: List, rename, merge, and delete tags.
- **Highlights**: Read and manage highlights from bookmarks.
- **Organization**: AI-powered suggestions for tags and collections.
- **Maintenance**: Scan library for broken links, duplicates, and untagged items.

## Usage

### Organization

"Analyze my recent bookmarks and suggest a collection for them."
"Suggest tags for this URL: https://example.com"

### Search

"Find all bookmarks from last month about 'TypeScript'."
"Search for bookmarks with tag 'recipe' and domain 'nytimes.com'."

### Cleanup

"Scan my library for broken links."
"Empty my trash."

## Configuration

Requires a `RAINDROP_ACCESS_TOKEN` from your Raindrop.io integrations settings: https://app.raindrop.io/settings/integrations
