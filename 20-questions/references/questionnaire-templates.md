# Questionnaire Templates by Config Type

## Terminal Emulators (Kitty, Alacritty, WezTerm, Ghostty)

**Round 1 — Basics:**
1. What shell do you use? (bash/zsh/fish/other)
2. Floating DE or tiling WM? Which one?
3. Normal window decorations or minimal/borderless?
4. Fixed window size or remember last session?

**Round 2 — Appearance:**
5. Font family and size preference?
6. Dark theme, light theme, or specific scheme (Dracula, Gruvbox, Catppuccin, Tokyo Night, custom)?
7. Cursor style: block, beam, underline? Blinking or solid?
8. Transparency level: none, slight (10-20%), heavy?

**Round 3 — Behavior:**
9. Mouse: hide after inactivity? Copy on select? Middle-click paste?
10. Scrollback lines: default (2000), more (10000), or unlimited?
11. Bell: audio beep, visual flash, or silent?
12. Confirm before closing if process is running?

**Round 4 — Power Features:**
13. Images in terminal (icat)?
14. Remote control / automation integration?
15. Shell integration (prompt jumping, last command output)?
16. SSH kitten for remote terminfo fix?
17. Startup session (auto-open tabs/windows on launch)?
18. Single-instance mode (new launches = tabs, not windows)?

**Round 5 — Keybinds & Navigation:**
19. Vim-style navigation between splits?
20. Direct tab jumping (Ctrl+Alt+1-9)?
21. Hints mode for URLs/files/hashes?
22. Custom keybinds for any specific workflow?

## Shells (Bash, Zsh, Fish)

**Round 1:**
1. Current shell and any framework (oh-my-zsh, prezto, fish builtin)?
2. Prompt style: minimal, powerline, starship, custom?
3. Color preference for prompt?

**Round 2:**
4. Favorite aliases you already use or want?
5. History size and behavior?
6. Tab completion style?

**Round 3:**
7. Plugins/tools: fzf, zoxide, bat, eza, ripgrep, fd?
8. PATH additions or custom env vars?
9. Sudo behavior fixes?

**Round 4:**
10. Keybinds: vim mode, custom shortcuts?
11. Auto-suggestions? Syntax highlighting?
12. Any shell integrations (kitty, tmux, etc.)?

## Editors (Neovim, Vim, Helix)

**Round 1:**
1. Color scheme preference?
2. Font and size?
3. Line numbers: absolute, relative, both, none?
4. Indentation: tabs or spaces? Width?

**Round 2:**
5. File tree / sidebar: nvim-tree, neo-tree, mini.files, none?
6. Fuzzy finder: telescope, fzf-lua, none?
7. Status line: lualine, mini.statusline, none?

**Round 3:**
8. LSP / language servers needed?
9. Completion engine: nvim-cmp, coc, builtin?
10. Git integration: gitsigns, fugitive, none?

**Round 4:**
11. Keybind philosophy: pure vim, IDE-like, custom?
12. Which key popup? Which-key or none?
13. Terminal inside editor? Toggleterm, none?

## Window Managers (i3, Hyprland, Sway)

**Round 1:**
1. Monitor count and layout?
2. Key modifier: Super (Windows), Alt, or custom?
3. Gaps between windows? Size?
4. Border style: pixel, normal, none?

**Round 2:**
5. Status bar: waybar, polybar, i3status, custom?
6. App launcher: rofi, wofi, dmenu, custom?
7. Notification daemon: dunst, mako, builtin?

**Round 3:**
8. Startup apps on login?
9. Window rules (float certain apps, assign workspaces)?
10. Scratchpad usage?

**Round 4:**
11. Animations: yes/no? Speed?
12. Blur / transparency for windows?
13. Wallpaper tool?

## Tmux

**Round 1:**
1. Prefix key: Ctrl+b, Ctrl+a, or custom?
2. Status bar: top or bottom? Style?
3. Mouse support: yes/no?

**Round 2:**
4. Pane navigation: vim keys (h/j/k/l) or arrows?
5. Pane splitting keybinds?
6. Window numbering: start at 0 or 1?

**Round 3:**
7. Copy mode: vim or emacs keys?
8. Clipboard integration?
9. Session management: manual, tmux-resurrect, tmux-continuum?

**Round 4:**
10. Plugins wanted (tpm, sensible, vim-tmux-navigator, etc.)?
11. Popup menus / fuzzy window switching?
12. Integration with terminal image protocols?
